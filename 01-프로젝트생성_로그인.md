# 프로젝트 설정
### 빌드 시스템 : Gradle (Groovy)
### JAVA : 23
### JDK : 23.0.2
### Spring Boot : 3.4.2
### 의존성 추가

- Spring Web : REST API
- Spring Security : 사용자 인증 및 보안
- OAuth2 Client : 소셜 로그인
- Spring Boot DevTools : 개발 시 자동 Reload
- Lombok : 어노테이션 기반으로 코드 자동완성
- Spring Data JPA : MySQL 연동
- MySQL Driver : DB 연동
- H2 Database : 개발용 인메모리 DB
